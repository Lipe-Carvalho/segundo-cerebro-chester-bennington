# Segundo Cérebro - Chester Bennington

## Contexto

Este projeto foi desenvolvido utilizando o NotebookLM como ferramenta de pesquisa e organização do conhecimento.

O tema escolhido foi Chester Bennington, cantor, compositor e vocalista da banda Linkin Park.

## Objetivos

* Estudar a trajetória de vida de Chester Bennington.
* Compreender seu processo criativo.
* Analisar seu estilo vocal.
* Identificar temas recorrentes em suas composições.
* Utilizar IA para organizar e consultar informações sobre o artista.

## Ferramentas Utilizadas

* NotebookLM
* GitHub
* Markdown

## Estrutura do Projeto

* fontes.md → Fontes utilizadas na pesquisa.
* prompts.md → Perguntas e testes realizados no NotebookLM.
* guia-estudo.md → Resumo final do aprendizado.
